#whatsForDinnerTO

A web application that helps you decide what you want to eat! Select an ingredient and geographic location, then recipes and Toronto restaurants will appear according to your specifications.

    Made with:
  • HTML, CSS, JavaScript
  • Firebase (user validation)
  • Edamam and Zomato API's
  • Deployed on GitHub Pages

HiNeighbour!

A social networking platform that helps neighbours connect with each other based on who is around their immediate geographical location. Users can create events and send invites, make posts, and comment on posts.

    Made with:
  • HTML, CSS, JavaScript
  • Passport.js (user validation)
  • Google Maps API
  • MySQL linked tables and Sequelize.js ORM
  • Deployed on Heroku

Budget+

A financial app that provides the user with a recommended budgeting plan based on their income. Users can link their bank accounts and receive rewards if their expenses match what they budgeted for.

    Made with:
  • HTML, CSS, JavaScript
  • React.js
  • MongoDB and Mongoose.js
  • Plaid API
  • Dual servers, one for a user validation API and the other for financial data
