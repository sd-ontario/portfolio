whatsForDinnerTO

Hosted on GitHub Pages

The first group project of the Coding Bootcamp at U of T School of Continuing Studies. The purpose of the application is to help users find their next meal. The user specifies the region of cuisine and the type of food, then they are given recipes and restaurants that match their serach criteria.

Technologies used:

  • Edamam API
  • Zomato API
  • Firebase and Parsely.js (for user validation)
  • JavaScript, jQuery, AJAX, HTML, CSS, Bootstrap

HiNeighbour!

Hosted on Heroku

The first "full stack" project of the course. HiNeighbour is a social networking site that allows users to effectively interact with their neighbours. You can discover who lives around you, see events that users post, create and hold your own events, and read posts from neighbours or post your own for alerts, inquiries, and discussion.

Technologies used:

  • Node.js
  • Express.js
  • Handlebars.js (for dynamic templating)
  • MySQL and Sequelize.js
  • Passport.js (for user validation)
  • Google Maps API

Budget+

Hosted on Heroku

Final project of the course. Budget+ is an application that helps users allocate their spending according to what is recommended by our algorithm. Incorporates the Plaid API which extracts bank transactions so users can compare their budgeting goals with their actual spending. The plan is to work with online retailers and provide incentives that reward users when their spending reflects their budgeting goals.

Technologies used:

  • MongoDB
  • Express.js
  • React
  • Node.js
  • Plaid API
  • Chart.js
  • bcrypt
